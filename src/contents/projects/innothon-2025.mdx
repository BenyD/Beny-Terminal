## Innothon 2025: Tech Fest Website

A website for the Tech fest cultural event organized by the Department of Computer Science and Engineering, Hindustan Institute of Technology and Science, Chennai. It provides information about the event, registration form and gives us an admin panel to manage the event.

### Features

- **Event Information:** Comprehensive details about the tech fest
- **Registration System:** Streamlined participant registration process
- **Admin Panel:** Complete event management interface
- **Responsive Design:** Accessible across all devices
- **Modern UI:** Engaging and user-friendly interface

### Stack

- [Next.js](https://nextjs.org) - React framework for building full-stack web applications
- [React](https://reactjs.org) - JavaScript library for building user interfaces
- [TypeScript](https://www.typescriptlang.org) - Typed JavaScript for better development experience
- [TailwindCSS](https://tailwindcss.com) - Utility-first CSS framework
- [Supabase](https://supabase.com) - Open source Firebase alternative
- [Framer Motion](https://www.framer.com/motion) - Production-ready motion library for React
- [Magic UI](https://magic-ui.com) - Modern UI components library
