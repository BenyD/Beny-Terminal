## Beny's CV Website: Professional Digital Presence

A clean and professional CV website designed to showcase my skills, experience, and projects. It offers a centralized digital resume for potential employers and collaborators.

### Features

- **Professional Portfolio:** Showcase of skills, experience, and projects
- **Interactive Interface:** Clean and intuitive user experience
- **Responsive Design:** Seamless experience across all devices
- **Modern UI:** Professional and contemporary design

### Stack

- [Next.js](https://nextjs.org) - React framework for building full-stack web applications
- [TypeScript](https://www.typescriptlang.org) - Typed JavaScript for better development experience
- [TailwindCSS](https://tailwindcss.com) - Utility-first CSS framework
- [Shadcn UI](https://ui.shadcn.com) - Re-usable components built with Radix UI and Tailwind CSS
- [Magic UI](https://magic-ui.com) - Modern UI components library
