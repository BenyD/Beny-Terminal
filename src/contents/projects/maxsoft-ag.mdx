## Maxsoft AG: Swiss IT Solutions

A corporate website for a Swiss IT company, showcasing their services and expertise. Provides an interactive and professional experience for potential clients.

### Features

- **Service Showcase:** Comprehensive display of IT services
- **Interactive Elements:** Engaging 3D and interactive components
- **Professional Design:** Corporate and trustworthy appearance
- **Client Engagement:** Clear call-to-actions and contact points

### Stack

- [Framer](https://www.framer.com) - Interactive design and prototyping tool
- [Figma](https://www.figma.com) - Collaborative interface design tool
- [Spline](https://spline.design) - 3D design tool for web
- [React](https://react.dev) - JavaScript library for building user interfaces
- [API](https://nextjs.org/docs/api-routes/introduction) - RESTful API integration
