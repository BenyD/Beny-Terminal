## RapidResume: ATS-Friendly Resume Builder

A resume builder for the Indian market with ATS compliance in mind. Users can create professional resumes in minutes with built-in ATS scoring.

### Features

- **ATS Compliance:** Built-in scoring and optimization
- **Quick Creation:** Resume generation in minutes
- **Professional Templates:** Industry-standard designs
- **Market Focus:** Tailored for Indian job market

### Stack

- [Next.js](https://nextjs.org) - React framework for building full-stack web applications
- [React](https://react.dev) - JavaScript library for building user interfaces
- [Redux](https://redux.js.org) - State management library
- [TypeScript](https://www.typescriptlang.org) - Typed JavaScript for better development experience
- [TailwindCSS](https://tailwindcss.com) - Utility-first CSS framework
