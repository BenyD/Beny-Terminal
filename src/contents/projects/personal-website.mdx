## Beny's Personal Website: Interactive Terminal Experience

An interactive terminal like personal website that provides insights into my professional journey and projects. It showcases my portfolio and personal brand in an engaging and immersive way.

### Features

- **Terminal Interface:** Command-line inspired interactive experience
- **Dynamic Content:** Real-time content updates and interactions
- **Portfolio Showcase:** Comprehensive display of projects and skills
- **Personal Branding:** Unique and memorable user experience

### Stack

- [Next.js](https://nextjs.org) - React framework for building full-stack web applications
- [TypeScript](https://www.typescriptlang.org) - Typed JavaScript for better development experience
- [TailwindCSS](https://tailwindcss.com) - Utility-first CSS framework
- [MDX](https://mdxjs.com) - Markdown for the component era
- [API](https://nextjs.org/docs/api-routes/introduction) - Next.js API Routes
- [Prisma](https://www.prisma.io) - Next-generation Node.js and TypeScript ORM
- [Supabase](https://supabase.com) - Open source Firebase alternative
