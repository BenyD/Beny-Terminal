## CGPA Calculator: Student Grade Management

A simple web app for students to calculate their CGPA accurately. Provides a clean and intuitive user interface for fast and reliable results.

### Features

- **Accurate Calculations:** Precise CGPA computation
- **User-Friendly Interface:** Clean and intuitive design
- **Quick Results:** Instant grade calculations
- **Responsive Design:** Works on all devices

### Stack

- [Next.js](https://nextjs.org) - React framework for building full-stack web applications
- [TypeScript](https://www.typescriptlang.org) - Typed JavaScript for better development experience
- [TailwindCSS](https://tailwindcss.com) - Utility-first CSS framework
- [Shadcn UI](https://ui.shadcn.com) - Re-usable components built with Radix UI and Tailwind CSS
